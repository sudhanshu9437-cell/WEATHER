# 🌦️ Weather App

A simple and responsive weather website that shows real-time weather data for any city using a weather API.

---
## 🌐 Live Demo

https://sudhanshu9437-cell.github.io/WEATHER/

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Displays temperature, humidity, and weather condition  
- ☁️ Dynamic weather icons (clear, cloudy, rain, etc.)  
- 📱 Responsive design (works on mobile & desktop)  
- ⚡ Fast API-based data fetching  

---

## 🛠️ Tech Stack

- HTML  
- Tailwind CSS  
- JavaScript (Fetch API)  
- Weather API (e.g., OpenWeatherMap)

---

## ⚙️ How It Works

1. User enters a city name  
2. App sends request to weather API  
3. Receives JSON data  
4. Displays weather details on UI
